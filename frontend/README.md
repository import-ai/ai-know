## Ai know FE

## how to run
1. copy `.env.example` to `.env`
2. make sure `corepack` is enabled
3. `yarn i`


## commands
1. `api:sync-json` - to download openapi schema json
2. `api:gen-swagger` - to generate swr hooks and ts types
